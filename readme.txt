// AutoConvertSetup for v3.3.0

- このファイルについて

AutoConvert用のセットアップファイルです。
AutoConvertに必要な実行ファイルやプラグインが入っています。
すべて32bitなので、必要に応じて64bitのものと入れ替えてください。
DGIndexNV、Comskip、logoGuilloは同梱しておりませんので各自設定の程お願いします。

- 使い方

ファイルをAutoConvertに上書きしてください。
また、Avisynthがインストールされていない場合はAviSynth_20160517_VC2008Exp SSE2.exeを実行してください。

- ライセンス

AutoConvertは、再頒布、同梱、改変等は自由です。
ただし、AutoConvertSetupに同梱されているものについては各ライセンスに従ってください。
もし、再頒布に問題がありましたら、 https://rndomhack.com/ または twitter:@rndomhack まで連絡のほどよろしくお願いいたします。
