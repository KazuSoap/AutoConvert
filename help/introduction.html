導入

Avisynthのプラグインは標準のプラグインフォルダではなく、AutoConvertのpluginsフォルダに入れてください。また、Avisynth関連のソフトウェア・プラグインはすべて32bitで揃えてください。

ソフトウェア

Avisynth

動画編集ソフトウェア(フレームサーバ)

Avisynth_258.exeまたはalpha版をダウンロード

インストーラに従ってインストール

avs2pipemod

avsファイルを標準出力に出力

設定の AutoConvert -> パス -> avs2pipemod に avs2pipemod.exe へのパスを記述

TsSplitter

MPEG-2 TSの分割など

設定の AutoConvert -> パス -> TsSplitter に TsSplitter.exe へのパスを記述

ffprobe

MPEG-2 TSを解析

正しいPIDが取得できないなどの場合に使用してください

設定の AutoConvert -> パス -> ffprobe に ffprobe.exe へのパスを記述

DGMPGDec

MPEG-2 TSからAACを分離, MPEG-2 Videoのデコード

設定の AutoConvert -> パス -> DGIndex に DGIndex.exe へのパスを記述

DGDecode.dll -> plugins\DGDecode.dll

DGDecNV

DGMPGDecのCUDA版(有料)

設定の AutoConvert -> パス -> DGIndexNV に DGIndexNV.exe へのパスを記述

DGDecodeNV.dll -> plugins\DGDecodeNV.dll

ts2aac

MPEG-2 TSからAACを分離

設定の AutoConvert -> パス -> ts2aac に ts2aac.exe へのパスを記述

ts_parser

MPEG-2 TSからAACを分離

設定の AutoConvert -> パス -> ts_parser に ts_parser.exe へのパスを記述

Comskip

CMカット

設定の AutoConvert -> パス -> Comskip に comskip.exe へのパスを記述

logoGuillo

CMカット

設定の AutoConvert -> パス -> logoGuillo に logoGuillo.exe へのパスを記述

logoフォルダに ロゴ名.lgd、ロゴ名.lgd.autoTune.param を配置

join_logo_scp

CMカット

本来3つのソフトですが、AutoConvert上ではjoin_logo_scpとひとくくりで扱います。

設定の AutoConvert -> パス -> logoframe に logoframe.exe へのパスを記述

設定の AutoConvert -> パス -> chapter_exe に chapter_exe.exe へのパスを記述

設定の AutoConvert -> パス -> join_logo_scp に join_logo_scp.exe へのパスを記述

logoフォルダに ロゴ名.lgd を配置

AvsPmod

avsファイル編集

設定の AutoConvert -> パス -> AvsPmod に AvsPmod.exe へのパスを記述

Caption2Ass

字幕の抽出

設定の AutoConvert -> パス -> Caption2Ass に Caption2Ass_PCR.exe へのパスを記述

AutoVfr

自動可変フレームレート

設定の AutoConvert -> パス -> AutoVfr に AutoVfr.exe へのパスを記述

Its.dll -> plugins\Its.dll

qaac

AACのエンコーダ

設定の プリセット -> (qaacを使用するプリセット) -> 音声 -> エンコーダーのパス に qaac.exe へのパスを記述

neroAacEnc

AACのエンコーダ

設定の プリセット -> (neroAacEncを使用するプリセット) -> 音声 -> エンコーダーのパス に neroAacEnc.exe へのパスを記述

x264

H.264のエンコーダ

設定の プリセット -> (x264を使用するプリセット) -> 映像 -> エンコーダーのパス に x264.exe へのパスを記述

QSVEnc

H.264のエンコーダ

設定の プリセット -> (QSVEncを使用するプリセット) -> 映像 -> エンコーダーのパス に QSVEncC.exe へのパスを記述

NVEnc

H.264のエンコーダ

設定の プリセット -> (NVEncを使用するプリセット) -> 映像 -> エンコーダーのパス に NVEncC.exe へのパスを記述

FFmpeg

エンコーダ

設定の プリセット -> (FFmpegを使用するプリセット) -> 映像 -> エンコーダーのパス に ffmpeg.exe へのパスを記述

L-SMASH

MP4のMuxer

設定の AutoConvert -> パス -> L-SMASH muxer に muxer.exe へのパスを記述

設定の AutoConvert -> パス -> L-SMASH remuxer に remuxer.exe へのパスを記述

設定の AutoConvert -> パス -> L-SMASH timelineeditor に timelineeditor.exe へのパスを記述

MP4Box

MP4のMuxer

設定の AutoConvert -> パス -> MP4Box に MP4Box.exe へのパスを記述

mkvmerge

MatroskaのMuxer

設定の AutoConvert -> パス -> mkvmerge に mkvmerge.exe へのパスを記述

rplsinfo

MPEG-2 TSから番組情報等を取得

EpgDataCapBonが作成する".program.txt"がない場合(TvTestなどで録画したときなど)に使用してください

設定の AutoConvertUtility -> 全般 -> rplsinfo に rplsinfo.exe へのパスを記述

設定の AutoConvertUtility -> 全般 -> rplsinfo使用 を チェック

SCRename

しょぼいカレンダーから情報を取得

設定の AutoConvertUtility -> 全般 -> SCRename に SCRename.vbs へのパスを記述

設定の AutoConvertUtility -> 全般 -> SCRename使用 を チェック
